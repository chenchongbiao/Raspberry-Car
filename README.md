# Raspberry-Car
树莓派小车

1. 采用双轮差速方式移动

2. 通过网络远程控制小车

3. 能够实时传送图像

4. 超声波测距

## 硬件

- 树莓派4B 2GB
- HC-SR04超声波测距模块
- TB6612FNG电机驱动
- N20减速电机
